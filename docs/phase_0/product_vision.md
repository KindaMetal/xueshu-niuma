# 产品核心概念画像 (Product Vision)

## 1. 核心产品定义 (One-Sentence Pitch)
一款伪装成 MBTI 心理学的**极其刻薄、带有强烈黑色幽默的“学术实验耗材/当代研究生”防破防定性评估器**。

## 2. 宏观商业/传播动机 (Why exist?)
本项目**不追求**任何专业心理学或职场指导意义。它唯一的存在价值，是为处于高压、焦虑、充满荒诞感的现代高校学术系统中的人群提供一个情绪宣泄口。通过“地狱笑话”级别的文案戳中对方的痛点并形成极强的共鸣感，使其成为各大高校寝室群、实验室微信群内部疯狂转发的“电子榨菜”。

## 3. 目标用户画像 (Target Audience)

### 🎯 **核心传播节点：当代研究生 (Master & PhD Candidates)**
*   **痛点特征**：常年被科研进度、盲审、组会、无脑实验消磨意志。
*   **情绪状态**：长期处于“明天就退学”与“再改一版”的薛定谔叠加态；极度容易情绪内耗。
*   **审美/幽默偏好**：喜欢极简的“死鱼眼”画风，抗拒大道理和鸡汤，对极具讽刺意味的发疯文学、“学术黑话”（如：画大饼、造数据、反向PUA）有天然好感。

### 🎯 **潜在破圈人群：部分大四保研/考研党、重压下的本科生**
*   **特征**：试图或者正在接触残酷的科研体系，对于“卷”和“绩点陷阱”深有感触。

## 4. 绝对非目标用户 (Negative Audience List)
本项目开发和文案必须**坚决屏蔽/无视**以下人群的感受或需求，绝不允许为了迎合这些人而稀释了产品的“学术阴暗味”：

*   🚫 **初高中生**（不理解科研、组会或发 Paper 的残酷性，会觉得测试词汇枯燥）。
*   🚫 **已进入社会的普通打工人/纯社畜**（早已经历过社会毒打，学术圈的伤痛对他们而言太遥远）。
*   🚫 **寻求正规学术或职业发展规划的用户**（本测试没有也不提供任何积极向上的建设性意见）。
*   🚫 **传统严肃心理学爱好者**（本测试无意在心理测量信效度上做哪怕一丁点的努力）。

## 5. 体验北极星指标 (North Star Experience)
用户在做题时的直观体验必须是：
**“这题是谁出的？他在我实验室安了监控吗？我怎么选都像是一条可怜的丧家犬？？？”**
