# 爆发传播与成功指标 (Success Metrics)

## 1. 核心传播机制拆解 (Virality Engine)
对于“学术伪MBTI”测试而言，真正的“病毒式传播”只会在一种特定场景下引爆——**群聊里的“找同类”与“甩锅”效应**。

*   **痛点共鸣传播**：用户看到某项极端结果（如：“不洗烧杯的绝对利己者”），立刻联想到实验室里的某个同门，从而将结果截图发入课题组小群，配文：“@张三，这不就是你吗？”
*   **自黑炫耀传播**：用户测出极其凄惨的人格原型（如：“常年对冲组会风暴的避雷神盾”），出于一种深层的自我感动与苦中作乐，将其分享至朋友圈。
*   **敌对属性激发**：测试结果底部必须带有“✅与你天生搭档”和“❌绝对相克”，以此激发用户的好奇心，拉着周围其他同学强行作答以“化验一下八字合不合”。

## 2. 定性成功指标 (Qualitative Indicators)
当我们在社交媒体和校园网络中观察到以下现象时，代表产品达到了最高成功标准：

*   **指标A（破圈程度）：** 在未主动投放的前提下，从发给 5 个同学的小样本测试，自发裂变穿透到了至少 3 个不同高校的跨导师课题群组。
*   **指标B（造梗能力）：** 玩家自发使用在结果页生成的专有“黑话判词”作为自己的社交标签（比如在小红书发帖“家人们谁懂啊，测出来是个 INFP 深夜emo实验耗材”）。

## 3. 定量数据预期 (Quantitative Thresholds - 若接入埋点)
假设我们后续为测试网页接入极简追踪分析：

*   **完赛率 (Completion Rate)**：由于题目极具场景代入感（题目即段子），用户的跳出率应该极低。要求 100% 的完赛率达到 `>80%`。
*   **截图保存率/分享触发率**：结果页核心雷达图和判词被长按保存/点击“分享”按钮的转化率应 `>60%`。
*   **病毒系数 (K-Factor)**：理想中一个完成测试的玩家平均能拉来 `>1.5` 个新玩家做题（即 K系数稳定大于 1）。

## 4. 传播底线防火墙 (PR Firewall)
*   测试绝不包含任何诱导分享、强制关注公众号的锁死设计。
*   测试中的所有梗只对学术/研究现象本身开展精准吐槽，决不涉及任何上升到违法、网暴具体导师姓名的人身攻击红线，以保证项目本身的纯被动娱乐属性免受审查风险。
