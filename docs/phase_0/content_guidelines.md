# 文本与发疯黑话基调规范 (Content Guidelines)

为了贯彻我们在 Phase 0 确立的“高校生存伪 MBTI”概念，必须对未来的所有策划大纲、题干、选项和结果页判词做出最严苛的风格与质量约束。以下是本专案唯一的绝对红线：

## 1. 核心大体调性：死鱼眼式的无机质幽默
*   **不掺杂任何说教**：系统判定玩家是一个“满口虚空理论的划水废物”时，绝对不能加上“希望你今后改进努力”这种毫无存在意义的心灵鸡汤。
*   **极度陈述事实的冷酷感**：用毫无起伏的语调读出最残忍的校园真相。所有的文案基调都应该是冷色调的（如同站在学术金字塔最底层的旁观者冷眼嘲笑自己）。
*   **用词高度具象化**：
    *   ❌ 错误示范：你觉得压力很大，不知道怎么办。
    *   ✅ 正确示范：面对电脑屏幕前全线飘红的代码报错和明天早上 8 点的组会，你的心率比 CPU 温度还要稳定，选择熟练地开始点外卖。

## 2. 必备高校/科研高频黑话库 (Mandatory Lexicon)
在造题造典的过程中，必须无死角铺满以下圈层密码：
*   **人物代称**：老板君、大导、二导、野生博后、野生师兄、嫡长子/嫡长女、实验小白鼠、免费壮丁。
*   **场景动作**：搬砖、被 PUA、画大饼、造轮子、跑模型、炼丹、杀老鼠、洗烧杯、洗胶、凑字数。
*   **进度与死线**：极限微操、盲审拉闸、延毕警告、期刊接收 (Accepted)、查重降重。

## 3. 绝对禁区雷区规范 (Strict Taboos)
*   🚫 **禁止真正触碰精神创伤区**：诸如“抑郁”、“轻生倾向”、“严重的校园暴力欺凌”属于不可碰触的真正心理红线。所有的破防都必须建立在夸张的、可以付之一笑的“轻度科研受害者”层面。
*   🚫 **禁止直接隐射政治与法律底线**：不要在题目中教唆作弊、学术造假被查到等触及法律底线的“黑幽默”，我们的幽默是擦边造数据的心虚，而非鼓励真实的学术不端操作教学。
*   🚫 **禁止脱离学术背景**：如果一道题目可以直接无缝替换成小学考试抓作弊，或者初入职场开早会，那么这道题就是不合格的——题目必须且只能发生在“高校象牙塔”里。
